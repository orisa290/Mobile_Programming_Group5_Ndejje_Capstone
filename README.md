## 🎯 Feature Set

| # | Feature | Description |
|---|---------|-------------|
| 1 | **User Authentication** | Login and Signup with email/password using Firebase Auth |
| 2 | **Report Lost Items** | Users can report lost items with title, description, category, location, and image |
| 3 | **Report Found Items** | Users can report found items to help owners locate them |
| 4 | **Browse Items** | View all lost/found items in a scrollable LazyColumn with filtering |
| 5 | **Item Details** | View full item information including location, date, and contact options |
| 6 | **Image Upload** | Upload item photos via ImgBB API integration |
| 7 | **Real-time Chat** | Instant messaging between users to coordinate item returns |
| 8 | **My Reports** | View and manage user's own submitted items |
| 9 | **Filter by Type** | Toggle between Lost, Found, or All items |
| 10 | **Dark/Light Mode** | Full Material 3 theming with dark/light mode support |
