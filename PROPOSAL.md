# CAPSTONE PROJECT PROPOSAL

**Project Title:** Lost & Found Hub - Ndejje Campus Connect

**Submission Date:** 14th April 2026

## 1. Problem Being Solved

Every day at Ndejje University, students lose valuable items including phones, laptops, ID cards, wallets, books, and other personal belongings across all four campuses. Currently, there is NO centralized system to report lost items or notify finders.

## 2. Target Users

**Primary Users:** All Ndejje University students across all four campuses (estimated 5,000+ students)

**Secondary Users:** University staff, security personnel, and faculty members
